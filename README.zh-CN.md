# Editable

editable是一个富文本编辑器框架，不再依赖~~**contenteditable**~~属性即可提供稳定的富文本编辑能力。

演示：https://docs.editablejs.com/playground

## 开发

```bash
# 安装
pnpm install

# 启动
pnpm dev

```
